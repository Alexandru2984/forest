# Code Forest
An organic 3D visualizer for GitHub repositories, built with Go (Backend) and Three.js (Frontend).

## Tech Stack:
- **Backend:** Go (Golang) for efficient parsing of Git trees and blazingly fast APIs.
- **Frontend:** Three.js / WebGL for 3D rendering and visual effects in the browser.

## Deployment Status
- Domain: `forest.micutu.com`
- SSL: Let's Encrypt via Certbot.
